# Horizontal Navbar

A clean and simple horizontal navigation bar built with HTML and CSS.

## Project Preview

![screenshot of horizontal navbar](screenshots/horizontal-navbar.png)

## Objectives
Create a functional navigation menu and practice layout alignment using basic CSS.

## Key Features
* **Horizontal Layout:** Links are arranged side-by-side for easy access.
* **Hover Effects:** Links change color and show an underline when you move your mouse over them.
* **Responsive Design:** Uses a flexible container that fits the screen width.
* **Clean Styling:** Rounded corners and a soft color palette for a modern look.

## Concepts Learned
* **Flexbox Basics:** Using `display: flex` to align list items horizontally.
* **List Styling:** Removing default bullet points with `list-style: none`.
* **Link Customization:** Removing default underlines with `text-decoration: none` and adding custom hover states.
* **Box Model:** Managing `padding`, `margin`, and `border-radius` to shape the menu.

## Tools and Technologies Used
* HTML5
* CSS3
