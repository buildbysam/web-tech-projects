# Colored Boxes

A simple web project that displays a grid of colorful boxes using HTML, CSS, and JavaScript.

## Project Preview

![screenshot of colored boxes project](screenshots/colored-boxes.png)

## Objectives
Create a clean layout and practice generating elements using JavaScript loops.

## Key Features
* **Organized Layout:** Uses a centered flexbox container to align items perfectly.
* **Automatic Generation:** Uses a loop to create nine boxes without writing each one manually in HTML.
* **Color Grouping:** Each set of three boxes features a distinct, vibrant color.

## Concepts Learned
* **CSS Flexbox:** Using `display: flex` and `gap` to space out elements.
* **JavaScript Loops:** Using `for` loops to repeat the creation of elements.
* **CSS Selectors:** Using ID selectors to apply different background colors to specific groups.

* HTML5
