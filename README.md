# Web Technology Project Showcase

A professional web application designed to archive and display various technical assignments completed during the Web Technology course. This project serves as a digital portfolio to track progress from basic layouts to functional web logic.

## Project Motive
The goal of this application is to provide a clean, organized space to explore project source code and visual previews. It demonstrates the practical application of foundational web tools like HTML, CSS, PHP, and JavaScript.

## Core Features
* **Project Directory:** A complete collection of course assignments organized for easy browsing.
* **Source Code Viewer:** A dedicated way to view the actual code files for each specific project.
* **Tech Stack Display:** Clear visibility of the tools and languages used for every assignment.
* **Responsive Design:** A user-friendly interface that works across different screen sizes.

## Technical Stack
**Framework/Tools:** Built with Next.js (React) and TypeScript, styled with Tailwind CSS

## Project Structure
All individual class assignments and projects are stored within the `projects/` directory for easy access and organization.

[**Browse Projects**](./projects)

## Learning Goals
* Master the building blocks of the web: HTML, CSS, PHP, and JavaScript.
* Build responsive interfaces that function on both mobile and desktop devices.
* Write clean, well-structured, and maintainable code.
* Understand how to turn design concepts into working web pages.
